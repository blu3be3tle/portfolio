Blad - A Blood Donation Platform

Blad is a blood donation platform with a robust user role system, ensuring distinct privileges for each role. Admins can view and edit all user information and donation requests, overseeing the entire platform. Volunteers can access donation request lists and update their statuses in real time, while donors can create new requests—specifying blood type, location, and urgency—and view only their own contributions for privacy.

The platform is built using React for a dynamic, responsive frontend, styled with Tailwind CSS for a visually appealing experience. The backend leverages MongoDB for efficient data storage, Express.js for scalable APIs, and JWT for secure authentication. Firebase Authentication ensures reliable user verification, and Axios facilitates seamless HTTP requests between the frontend and backend. The entire application is deployed on Firebase, ensuring accessibility and reliability.
